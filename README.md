# CivicSense

A simple platform for people to report local issues in their neighborhood, such as potholes, broken amenities, or illegal dumping. 

Built with Next.js and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- Browse recent local reports
- Submit new reports with categories and locations
- Simple, clean interface
